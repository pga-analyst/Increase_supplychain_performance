# Increase_supplychain_performance

Quick analysis with on how to increase supply chain performance in a warehouse utilizing KPI to dictate staffing based on weighted averages with reliability scores that can be associated to HR data. Warehouse staffing in a supply chain org can significantly improve with metrics such as ship times, replenishment time or shipping accuracy when people are assigned work based on their ability to perform in their top respective area. Machine learning can be further applied to identify worker trends.
